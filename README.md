# Sistema de Gestión de Biblioteca

Aplicación de escritorio desarrollada en C# utilizando Windows Forms.

## Funcionalidades

- Gestión de libros (CRUD)
- Gestión de usuarios (CRUD)
- Registro de préstamos
- Buscador de libros y usuarios
- Control de inventario de libros
- Estadísticas con gráficas

## Tecnologías utilizadas

- C#
- Windows Forms
- .NET
- Git

## Estructura del sistema

El sistema está dividido en tres módulos principales:

### Libros
Permite registrar, editar, eliminar y buscar libros dentro del inventario de la biblioteca.

### Usuarios
Permite registrar y administrar los usuarios que pueden solicitar préstamos.

### Préstamos
Permite registrar préstamos de libros y visualizar estadísticas como:

- Libros más prestados
- Usuarios más activos

## Instalación

1. Clonar el repositorio
2. Abrir el proyecto en Visual Studio
3. Ejecutar la solución

## Uso

1. Registrar libros en el sistema
2. Registrar usuarios
3. Crear préstamos
4. Visualizar estadísticas en el menú principal
